当社は、任意にWEB買取サービスの内容を定め、理由の如何を問わずお客様の承諾なくして、会員サービスの内容の一部または全部の変更、追加及び廃止をすることができるものとします。

第２章 商品情報

１.査定金額表示について

査定金額の表示は消費税総額表示としています。

２.査定金額について

当サイト上の査定金額と、当社が運営する店頭ショップでの査定金額は異なった金額となる場合があるものとします。
当サイト上の査定金額では、同商品であっても買取金額が異なる場合があるものとします。
査定金額は、季節環境、相場動向、在庫状況等により、常に変動するものとします。
当サイトにて事前に査定金額を提示することがございますが、状態等により査定金額が変動したり、買取できない場合があるものとします。
品物が当社に到着しない限り、査定金額を提示することはできないものとします。
当サイトの査定金額表示について単品での査定金額が500円以上の物につきましては1品ごとでの査定結果の提示、 500円未満の物につきましては衣料（洋服等）・服飾雑貨（バッグ・靴・アクセサリー等）という表記で纏めて査定結果の提示をさせていただくものとします。
纏めて提示した物につきましては個別での返品等は受け付けないものとします。
査定結果に基づく当社からの提示金額に対して、会員は、承認もしくはキャンセルの選択権のみ有するものとし、金額の交渉や不服申し立てなどは一切行わないものとします。
尚、会員がこれに反して、交渉等を行ったとしても、当社はこれを一切受け付けないものとします。査定金額がつかなかった品物についても同様とします。

３.商品情報について

当社は、当サイト上の商品説明をできる限り正確に表示するよう努めるものとします。
商品画像はイメージであり、テキストの商品情報と異なる場合、テキストの商品情報が優先するものとします。
商品関連情報の表示は、当社が運営に関わる店頭ショップでの取り扱いを保証するものではないものとします。
当社で定める取扱アイテムの基準は、あくまで当社独自の観点によるものとします。
これらの基準が定める詳細は、社外秘事項であり、お客様へ情報を開示することはできないものとします。

お客様は、本規約に従って、本サービスをご利用するものとします。

    • 未使用品や同一商品を複数台お申し込みの場合、入手経路を確認させていただく場合がありますので、あらかじめご了承ください。

    • 動作確認を行いますので、必ずバッテリーが充電されている状態にしてください。万が一、充電されていない商品をお送りいただいた場合は、買取不可となります。
    • 動作確認を行いますので、暗証番号を次の初期設定番号に変更してください。docomo→0000　au→1234　SoftBank→9999 万が一、変更されていない商品をお送りいただいた場合は、買取不可となります。
    • 遠隔ロック（docomo「おまかせロック」　au「安心ロック」　SoftBank「安心遠隔ロック」）を設定している場合は、必ず解除してください。
    • 万が一、解除していない商品をお送りいただいた場合は、買取不可となります。
    • かんたんログインなどは、商品自体にログイン情報が登録されている場合がありますので、必ず削除や解除を行ってください。万が一消去されていないログイン情報などがあり、売却後、第三者の手に渡ったとしても、当社はその責を一切負わないものとします。
    • お申込み時に、ご利用者が買取可能商品に含まれると想定された商品であっても、査定の結果、当社基準により買取可能商品の対象外となる場合があります。

      • イ：万が一、買取完了後に残っていた分割払いが滞る等の理由によって、商品が使用不能となった場合は、当社に買取金額を全額返金すること。

      • エ：アを遵守し、イ・ウについて、なんら異議申し立てをしないこと。
    • 買取希望商品のうち、各キャリアのウェブサイト上でネットワーク利用制限が「×」または「-」等と表示され、申込時点で利用不可能になっている端末については、状態にかかわらず（新品同様であっても）買取不可となります。
